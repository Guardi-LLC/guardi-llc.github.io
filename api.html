Guardilua API

Welcome to the Guardilua API documentation. Use the endpoints below to integrate Guardilua's protection system into your projects.

Endpoints

  • POST /api/auth/register - Register a new user
  • POST /api/auth/login - Authenticate a user
  • GET /api/scripts/{id} - Retrieve a protected script
  • POST /api/scripts - Upload and obfuscate a script
  • GET /api/user/{id} - Fetch user details
